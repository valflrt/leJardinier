Introduction

"Le Jardinier" in french means The Gardener in english, it is a simple multi-task discord bot.

Commands

  • lj!help : Help command, gives the list of all the commands
  • lj!hey : Used to greet the bot (it also greets you back)
  • lj!t/f [?sentence] : Answers "true" or "false" randomly, includes sentence in the message if given
  • lj!percentage [?sentence] : Gives a random percentage, includes sentence in the message if given
  • lj!morse [sentence] : Encodes sentence to morse

Notes:

"lj!" is the prefix, the bot scans all messages that starts with it.

Words between square brackets in the command list are arguments, it means that text after a command in a message will be used in the command.

Example:

  • "[?sentence]" means you can add text after the command (optional)
  • "[sentence]" means you must add text after the command (required)
