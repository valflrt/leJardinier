Introduction

"Le Jardinier" in french means "The Gardener" in english, it is a simple multi-task discord bot.

How to use

1/ Command calls

First, you need to know how to call a command.

To do so, you will need to write a message that looks like:

[prefix][command namespace] [parameters]

Here are some examples:

  • lj!hey
  • lj!morse.encode this is an example

2/ Hierarchy system

I use a specific command hierarchy system: a "dotted" hierarchy.

That is to say the whole "path" to a command is called a command namespace and is composed of command identifiers separated by dots:

[command name].[subcommand name].[sub-subcommand name].[...]

Examples:

  • morse.table is the namespace for the subcommand "table" of the command "morse"
  • music.add.searchis the namespace for the sub-subcommand "search" of the subcommand "add" of the command "music"
